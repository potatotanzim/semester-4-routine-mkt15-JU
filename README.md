# JU Marketing Class Schedule

A fun, interactive, cartoon-themed daily class schedule for Jahangirnagar University (JU) Marketing Department, 2nd Year, 2nd Semester.

## Features
- **Dynamic Layout:** A responsive, visually engaging schedule broken down by days of the week.
- **Cartoon Characters:** Includes fun animations of classic cartoon characters (Peter, Stewie, Quagmire).
- **Interactive Speech Bubbles:** Click on characters to see randomized fun quotes!
- **Pure HTML/CSS/JS:** No external libraries needed (other than Google Fonts). Just open the `index.html` file in any modern web browser.

## How to Use
1. Clone or download this repository.
2. Open `index.html` in your web browser.
3. Enjoy your class schedule!
